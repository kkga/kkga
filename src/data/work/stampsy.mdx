Stampsy was a social platform for sharing and discovering visual stories. I
served as a product designer and front-end engineer from 2012 to 2014, my first
full-time startup role after freelancing.

<div class="col-[breakout] grid grid-cols-subgrid">
  ## Stampsy Web
  <time datetime="2014" class="aside">
    2014
  </time>
</div>

As a key team member, I contributed to product strategy, UI/UX design, and
front-end development. I prototyped core interactions using Framer.js (at that
time it was just an open-source project by Koen Bok) and established a
responsive CSS architecture, implementing the design system as reusable React
components.

  ## Stampsy iPad app
  <time datetime="2012" class="aside">
    2012
  </time>
</div>

The platform originated as an iPad application in 2012, focusing on intuitive
story creation through touch-based interaction. I led the design and prototyping
of the core user experience.
