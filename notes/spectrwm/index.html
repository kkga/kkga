Some notes I'm collecting while setting up a comfortable desktop with my new favorite window manager

Updated on: 15 Nov 2020  · 
  • #note
  • #software

what I like

  • overall “lightweight-ness” and simplicity;
  • the master/stack model works pretty much like dwm, which is perfect;
  • great set of options, so achieving the same workflow I had on dwm is easy;
  • plain text configuration file;
  • single tag-set across multiple monitors;

don’t like

  • the built-in bar is great, but lacks a system tray;
